Microsoft Copilot

Word・Excel・Outlook・Teams など、普段のMicrosoft 365アプリの中で使えるAIアシスタント。

公式サイトを開く →
1

サービス概要・特徴

Microsoft Copilot（旧 Bing AI / Microsoft 365 Copilot）は、Microsoftが提供するAIアシスタント。Word・Excel・Outlook・Teams など Microsoft 365 アプリに組み込まれているのが最大の特徴で、普段の業務アプリの中でそのままAIを呼び出せます。Webチャット版（無料）もあります。

🧩 Microsoft 365に統合 🪟 Windowsにも標準搭載 🔎 Web検索（Bing）連携 🖼️ 画像生成にも対応
2

使い方のポイント（プロンプト例）

「どのアプリの・何を・どうしたいか」を伝えると、各アプリの中で効率よく使えます👇

📊 Excelの数式

「この売上表に、月別合計と前月比を出す数式を入れて。グラフも作って。」

🗒️ Teamsの議事録

「この会議の文字起こしから、決定事項とToDo（担当つき）を箇条書きでまとめて。」

📧 Outlookのメール

「このスレッドを3行で要約して、丁寧な返信案も作って。」

3

Word・Excel・Teams・Outlook での活用

普段使う Microsoft 365 アプリの中で、そのままCopilotを呼び出せます（Microsoft 365 Copilot 契約時）。

📝

Word

文章のたたき台作成・要約・体裁の調整。

📊

Excel

数式の作成、集計・分析、グラフの自動生成。

🎥

Teams

会議の自動要約・メモ作成、ToDo抽出。

📧

Outlook

長いメールの要約、返信文の下書き作成。

📽️

PowerPoint

プレゼン資料のたたき台作成・スライド構成の提案。

📓

OneNote

メモの整理・議事録のまとめ・アイデアの構造化。

4

中小企業での主な活用シーン

🗒️

議事録の自動化

Teams会議をそのまま要約し、決定事項とToDoを整理。

📈

Excel数式の生成

「やりたいこと」を言うだけで集計・分析の数式を作成。

📧

メール要約・返信

Outlookの長いスレッドを要約し、返信案も用意。

📑

資料作成

Word・PowerPointで報告書や提案資料のたたき台を作成。

🌐

翻訳・多言語対応

海外取引先へのメールや資料を自然な英語に翻訳。

🔍

チェック・校正

文章の誤り・抜け漏れの確認、表現の改善提案。

5

他ツールとの使い分け

Microsoft Copilotは「Microsoft 365（Office）を使う職場」に最適。Word・Excel・Teamsの中で完結できます。Google中心なら Gemini、まず万能に試すなら ChatGPT、長文・丁寧さなら Claude が得意です。
