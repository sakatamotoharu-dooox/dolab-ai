> 中小企業向け AI 活用プラットフォーム

### キャッチコピー
**自分で学んで、自分でAIを動かす。**

外注も高額SaaSも不要。Do Lab AI は、業務改善AIを自分で作り・動かし・改造できる人を育てます。

- 設計書Bot（M1）→ `/m1/`

## M1 — 設計書Bot（`/m1/`）

### キャッチコピー
**Claudeに話しかけると設計書が出てくる**

業務の内容を入力するだけで、AI化のための設計書（MD）が自動生成されます。基礎版・選択式版の2パターンを用意しています。

### プラン

#### 基礎版
こんな人向け：AI・設計書が初めての方

自由に入力できるシンプルな設計書Bot。「どんな業務か」「何をAI化したいか」を自分の言葉で話しかけるだけ。
→ Claude Projectsを開く

#### 選択式版
こんな人向け：業務が決まっている方

業種・業務・AIレベルを選ぶだけで分岐して最適な設計書を生成。社労士向けは給与計算・社会保険・労務相談の3パターンに対応。
→ Claude Projectsを開く

### 次のステップ
設計書ができたら次はM2へ。設計書をもとに、すぐ動かせるAI指示文セット（AIパック）を受け取れます。
→ M2 AIパックを見る

## 無料相談・応募（`/apply.html`）

> 無料相談フォームは外部Googleフォームへ遷移します。

### キャッチコピー
**まず話してみてください。一緒に考えます。**

「AIを使ってみたいけど何から始めればいい？」そんな段階でも大丈夫。現状をお聞きして、最適なステップをご提案します。

### こんな方に向いています
- AIを業務に活かしたいが何から始めるかわからない
- 外注せず自分・社内でAIを動かせるようになりたい
- ChatGPTは使っているがもっと深く活用したい
- 設計書・指示文を自分で作れるようになりたい
- 中小企業・士業・個人事業主として業務を効率化したい
- 特命AI集中道場・特命AI室に興味がある

### 応募後の流れ
1. **フォーム送信** — 現状や課題を入力
2. **日程調整** — 2営業日以内にご連絡
3. **無料相談** — オンライン30分
4. **プラン提案** — 最適なステップをご案内

### フォーム項目
| 項目 | 必須 |
|------|------|
| お名前 | ✓ |
| 会社名 / 屋号 | — |
| メールアドレス | ✓ |
| 業種・職種 | ✓ |
| 興味のあるサービス | — |
| 現状の課題・相談したいこと | ✓ |

送信後、2営業日以内にメールでご連絡します。
